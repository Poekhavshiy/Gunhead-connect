1. System Requirements
-----------------------
- Operating System: Windows 10 or later (64-bit)
- Minimum RAM: 4 GB
- Disk Space: 100 MB
- Dependencies: 
  - Microsoft Visual C++ Redistributable (if required)
  - Internet connection for API interactions

---

2. Installation Instructions
-----------------------------
1. Run the `KillAPIConnectPlusSetup.msi` installer.
2. Follow the on-screen instructions to complete the installation.
3. By default, the application will be installed in:
   `C:\Program Files\KillAPI Connect Plus\`

---

3. Application Features
------------------------
- **Log Monitoring**: Real-time monitoring of game logs.
- **Theme Selection**: Customize the application's appearance with multiple themes.
- **Language Support**: Choose from a variety of languages, including English, Deutsch, Espa√±ol, and more.
- **API Integration**: Seamless interaction with the KillAPI service.
- **Notifications**: Get alerts for important events.

---

4. Usage Instructions
----------------------
1. Launch the application from the Start Menu or desktop shortcut.
2. Configure the game directory and API key in the **Settings** window.
3. Click **Start Monitoring** to begin tracking game logs.
4. Use the **View Log** button to see real-time log updates.
5. Customize the application using the **Change Theme** and **Change Language** options in the Settings.

---

5. Troubleshooting
-------------------
- **Missing DLLs**: Ensure all required DLLs are in the application directory. If issues persist, reinstall the application.
- **API Key Issues**: Verify your API key in the **Settings** window.
- **Log File Not Found**: Ensure the correct game directory is configured in **Settings**.
- **Language Not Changing**: Restart the application after changing the language.

---

6. Uninstallation Instructions
-------------------------------
1. Open the **Control Panel**.
2. Navigate to **Programs and Features**.
3. Select **KillAPI Connect Plus** and click **Uninstall**.
4. Follow the on-screen instructions to remove the application.
